# Data Analysis Notebooks 📊

This repository contains a curated collection of Notebooks from academic and self-learning projects focused on data analysis and environmental topics.

## 📚 About the Projects

- Projects labeled with `-tuni` were developed at **Tampere University (Finland)** as part of my Master's program in Environmental Engineering, and are accompanied by a PDF report summarizing methods, insights, and references.
- The project labeled with `-unam` was completed during a data analysis course at **UNAM** (Mexico), and includes explanatory content in Spanish.
- All code is written in Python using Google Colab Notebooks.

## 📝 Notes

- Projects labeled `-tuni` were created in an academic setting at Tampere University.
- The UNAM notebook contains Spanish commentary in both code (`#`) and markdown cells, and some dataset column names and labels are also in Spanish. However, the structure and logic of the code remain accessible regardless of language.
- The datasets for the `-tuni` projects were sourced from official public/governmental repositories in Mexico City and New York City. For full source details, see the references section in each associated PDF report.
- The dataset used in the `-unam` notebook was provided directly by course instructors for learning purposes only.
